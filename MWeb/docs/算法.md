# 两数之和
## Python

```
Class Solution:
    def twoSum(nums, target):
        hash_map = dict()
        for i, x in enumerate(nums):
            if target - x in hash_map:
                return [i, hash_map[target - x]]
            hash_map[x] = i 
```
